Scripts for generating printable spice labels.

# Scripts

Generate all labels:

```powershell
./scripts/create-labels.ps1
```

Merge them into printable pages:
```powershell
./scripts/merge-labels.ps1
```

Merge into single pdf file:
```powershell
./scripts/merge-pages.ps1
```
